Manhattan Map Fundamentals

Layer Selection: Select first the base layer and then the attraction layer(s) from the Map Layer Selection tree on the right

Base Layers: OpenStreetMap and MapQuest are the two base layers offered for selection. The latter is set as default.

POIs: Select the attraction of your choice - Museums, Theatres, and/or Parks & Buildings from the Map Layer Selection tree

Info Popup: If you click a feature (corresponding to an attraction in the Map Layer Selection Tree), a popup appears indicating the name, address, and website of the attraction. Click on the (X) or any other attraction to make the current popup disappear.
