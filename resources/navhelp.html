Navigating through Manhattan

First click the button 'Place Markers' to initialize markers denoting start and destination points. The start point is always colored in green and (intermediate) destination points in red.

Once the start and (intermediate) destination points are defined, click the 'Draw Route' button to display the overlay route. Note the vertices are calculated to the nearest edge as defined in shortest path pgRouting wrapper.

To start anew, click the 'Clear Route' button. This clears markers and route from map. Note that clicking the 'Place Marker' button also clears an existing route.
